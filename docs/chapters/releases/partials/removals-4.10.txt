========================
Removals in version 4.10
========================

- Removed Django admin interface.
- Removed Django admin and sites apps.
- Remove setting `COMMON_COLLAPSE_LIST_MENU_OBJECT`.
- Remove setting `COMMON_COLLAPSE_LIST_MENU_LIST_FACET`.
- Remove support for user themes.
- Removed `elasticsearch-dsl` library. Its functionality is now merged into
  `elasticsearch`.
- The `safety` Python package and its related makefile targets.
