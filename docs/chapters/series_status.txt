Support levels
==============


Supported
---------

Fully supported indicate comprehensive support is available. This includes
technical assistance, bug fixes, security patches, and updates for the
latest features. Technical support will be provided to resolve any issues.

This series also receive translations updates.


Extended Support
----------------

During the extended support phase, security patches, bug fixes, and porting
of features are available only through sponsored or commercial support.

This series does not receive translations updates and online translation
files are deleted.


End of Life
-----------

End of life signifies that all support for the series has been discontinued.
Consequently, bug fixes and security patches will no longer be provided.
New versions of the series may be released only under exceptional
circumstances. Compatibility with the latest technologies is unlikely.
